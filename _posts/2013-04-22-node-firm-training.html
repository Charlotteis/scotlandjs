Pedro Teixeira and Nuno Job of the Node Firm will be providing a Node.js and Angular.js Workshop before ScotlandJS on Wednesday the 8th of May in TechCube!

Arrive at 9am for a 9:30am start, and running until around 4:30pm.

The workship will cover,

  • Serving Static files Using Node.js
  • Using Node.js as a RESTful web service
  • Connecting Node.js to CouchDB
  • Consuming a RESTful service using Angular.js
  • Using Angular.js routes
  • Session Management in Node.js
  • Connecting Node.js to Angular.js using Websockets
